1. Создаём базу данных
____________
use products
____________


2. Вставить 4 документа по товарам на сайте
   Атрибуты: 1.Название, 2.Категория (2 товара из одной категории, 2 товара из другой), 3.Цена, 4.Количество товаров на складе
_________________________________________________________________
db.products.insertMany([
{name: "говядина", category: "мясо", price: 700, count: 25},
{name: "курица", category: "мясо", price: 300, count: 50},
{name: "яблоки", category: "фрукты", price: 200, count: 200},
{name: "апельсины", category: "фрукты", price: 90, count: 300}
])
_________________________________________________________________


3. Рассчитать остаточную стоимость товаров в каждой категории (сумма цены, умноженной на остаток)
_________________________________________________________________________________________________
db.products.aggregate(
   [
     { $project: { name: true, category: true, total: { $multiply: [ "$price", "$count" ] } } }
   ]
)
_________________________________________________________________________________________________


4. Уменьшить количество товара на 1
____________________________________________________________
db.products.update({name : "говядина"}, {$inc: {count: -1}})
____________________________________________________________


5. Вывести top-2 самых дорогих товара
___________________________________________
db.products.find().sort({price:-1}).limit(2)
___________________________________________
